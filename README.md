# Тесты для BooksCollector

В этом проекте реализованы тесты для класса `BooksCollector`.

## Проверяемые функции

1. **Добавление книги**
   - Добавление новой книги.
   - Проверка, что нельзя добавить книгу дважды.
   - Проверка ограничения длины названия (не более 40 символов).

2. **Жанры книг**
   - Установка жанра для книги.
   - Проверка, что недопустимый жанр не устанавливается.
   - Получение жанра книги.
   - Получение всех книг определенного жанра.
   - Получение книг, подходящих для детей (исключая "Ужасы" и "Детективы").

3. **Избранные книги**
   - Добавление книги в список избранных.
   - Удаление книги из списка избранных..
